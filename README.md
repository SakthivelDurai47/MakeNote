# MakeNotes

## a simple CRUD application to make notes

---

### Features

- User Registration & Login
- User Specific Notes
- Markdown support
- Edit and Update notes
- Pin Notes to top
- Change different Themes based on Mood
- RateLimiting to stop malicious attacks

---

### Tech Stack

- React , TailwindCSS with DaisyUI, Lucid-React Icons - Frontend
- NodeJS, Express, Axios, Upstash (for Rate limiting) - Backend
- MongoDB - Database
